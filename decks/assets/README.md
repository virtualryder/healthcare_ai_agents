# Deck assets

**Optional official AWS logo.** The deck generators look for `aws-logo.png` in this folder. If you
drop the official AWS logo here (download it from the AWS brand portal / an internal AWS source),
every deck and one-pager uses it automatically in the top-right. If it is absent, the generators
render a clean **"aws" / "Built on AWS"** wordmark cue instead — so the decks are presentable as-is
and become fully on-brand the moment the official asset is added. (The artwork is intentionally not
bundled here to respect AWS trademark guidelines.)
