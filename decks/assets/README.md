# Deck assets

**No AWS logo is bundled here, and none should be added for customer-facing use.** See
[`../../BRAND-AND-TRADEMARK.md`](../../BRAND-AND-TRADEMARK.md) for the two-track rules.

- **Customer-safe / public track (default):** the generators render a plain-text **"Built on AWS"**
  wordmark cue — no AWS logo. This is the correct output for any customer-forward or public use.
- **Internal AWS track only:** an `aws-logo.png` in this folder will be picked up automatically, but
  the AWS logo may be used **only** inside AWS-approved internal templates, for AWS-internal audiences,
  and only with the required approval. Do **not** add the AWS logo to anything a customer or the public
  will see without written AWS approval. The artwork is intentionally not bundled here to respect AWS
  trademark guidelines.

When in doubt, ship without the logo and keep the "Built on AWS" plain-text cue.
